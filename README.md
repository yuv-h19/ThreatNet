# 🛡️ ThreatNet – Cyber Threat Intelligence Platform

**ThreatNet** is a collaborative Cyber Threat Intelligence (CTI) platform designed for the secure sharing and analysis of indicators. It provides a centralized terminal for analysts to report, track, and visualize IOCs, CVEs, and APT campaigns in real-time.

## 📌 Overview

ThreatNet is a **collaborative intelligence hub** designed for cybersecurity professionals — including SOC analysts, CTI researchers, MDR teams, and independent investigators. 

In the modern landscape, security teams must move from reactive incident response to proactive threat mitigation. ThreatNet enables this shift by centralizing structured intelligence reporting, search, and knowledge correlation within a single, secure terminal.

### **Key Capabilities:**
* **🔍 Indicator Sharing:** Submit and search structured Indicators of Compromise (IOCs) like IPs, Domains, and Hashes.
* **🛠️ Vulnerability Intelligence:** Report and query CVE data with product-based lookup capabilities.
* **🎯 APT Database:** Document and explore Advanced Persistent Threat (APT) groups and their known TTPs.
* **🏆 Gamified Analyst Workflow:** Earn contribution points through a rewards system designed to incentivize high-quality intelligence sharing.

---

## 🚀 Core Features

* **Structured Intelligence Engine:** Purpose-built forms for IOC, CVE, and APT reporting.
* **Tactical Analytics Dashboard:** Visual distribution of attack vectors including Phishing, Malware, and Ransomware.
* **Advanced Risk Calculator:** A multi-factor incident assessment model based on asset criticality and data sensitivity.
* **Secure Authentication:** Session-based analyst login and registration system.
* **Cyber-Themed UI:** A responsive, dark-mode interface optimized for SOC environments.



---

## 🏛️ System Architecture

The platform is built on a robust, scalable foundation:

* **MVC Design Pattern:** Clear separation of concerns between Models, Views (EJS), and Controllers.
* **Server-Side Rendering (SSR):** Dynamic content delivery using EJS templates for real-time data updates.
* **Relational Database:** MySQL integration for structured storage of complex threat correlations.
* **Modular Routing:** Scalable Express.js routing structure for efficient request handling.

---

## 🛠️ Technology Stack

* **Backend:** Node.js, Express.js
* **Frontend:** EJS (Embedded JavaScript), CSS3 (Custom Variables/Cyber-Glow), Vanilla JS/jQuery
* **Database:** MySQL
